How To Use The Template:

1. You need To Install Font, Please find a link above, download the free font and install to your computer.

2. To add or change the image.

  • (Powerpoint) Please delete the Red Placeholder Image by click and delete, then click the icon in the middle to browse your image. adjust the image by send it back if necessary.
  • (Keynote) Click the image icon on the right downside the image placeholder, to browse your image. adjust the image by send it back if necessary.
  • (Google Slide) Right click on the Red Placeholder Image and then Change Image
